# Project Overview

## プロジェクト名

**nyoice**

## 概要

nyoice は、AI Company が開発するトイレ管理ゲームです。  
Unity を用いて制作し、ChatGPT が作成する設計をもとに、開発 AI「Codex」が実装・検証・保守を担当します。

## 目的

プレイヤーがトイレ施設の運営と管理を行うゲーム体験を構築します。  
詳細なゲームルール、対象プラットフォーム、ビジュアル方針、リリース要件は、今後共有される設計資料を正として段階的に定義します。

## 開発体制

- **設計:** ChatGPT（鶫・ブチャラティ・遊星）
- **実装:** Codex
- **リポジトリ管理:** GitHub
- **ゲームエンジン:** Unity

## 現在のスコープ

現時点では、リポジトリとプロジェクト文書の基盤を整備しています。  
ゲーム仕様や技術要件は、設計資料の追加に合わせて本書および関連文書へ反映します。

## 開発方針

1. 共有された設計資料を実装上の基準とします。
2. 仕様が不明な箇所は推測で固定せず、未決定事項として明示します。
3. Unity の機能は責務ごとに分離し、変更しやすくテスト可能な構成を目指します。
4. 実装変更には、目的が分かるコミットメッセージと必要な文書更新を伴わせます。
5. 動作確認可能な単位で段階的に開発を進めます。

## ドキュメント方針

設計、仕様、技術判断、セットアップ手順は `Docs/` 以下で管理します。  
本書はプロジェクト全体の入口として維持し、詳細は個別の文書へ分割して参照します。
